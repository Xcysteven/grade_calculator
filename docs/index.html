UCSD Grade Dashboard

A Chrome extension for estimating UCSD DSC course grades from Gradescope rows and public course grading policies.

What It Does

  • Reads visible assignment names, scores, and max scores from Gradescope.
  • Detects supported DSC grading policies from public course pages.
  • Shows local current-grade estimates and category averages.
  • Supports local what-if edits for scores, weights, categories, and dropped items.

Important Limitations

Grades shown by this extension are estimates, not official course grades. Users should verify results against official syllabi and instructor announcements.

Privacy

Grade data is stored locally in the user's browser. Read the privacy policy.
